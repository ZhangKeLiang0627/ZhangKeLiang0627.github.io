杂记丨3月：枯木逢几春

杂记丨3月：枯木逢几春

Author：@kkl

写在前面

枯木逢几春？

嗨！大约过了多久？距离上一次文章发布确实有一段时间了，说实话今天是二六年三月十六日的凌晨，非常凌晨，我打算写到四点左右就去睡了，早上九点还得爬起来上班。这些日子我早已习惯，一面把支离破碎的工作流程梳理清晰，一面感慨新鲜感的消逝…

我确实有过离职的想法（呵，仅此一回，那是因为…噢，对了，我从前应该提起过吧，关于我又AKA救火队大队长的这件事情，似乎LD有这种惯性，觉得某个人好用了，就各个地方都放去用。所以，不难猜出，我又被“流放荒野”了，这是第四次。我并没有说这是不好的事情，对于一个初出茅庐的职场小大人，多学学更加横向的事情总归是好的，但我又在担忧这样的工作性质，日后会不会脏了简历，面试官一句你在前司担任的主要职务是？我支支吾吾说不清楚，最后从牙缝里挤出一个“全都干过”…这种…怎么听都像是骗人的鬼话（无奈

总不能在我刚对一个模块的工作稍微有些熟悉，可以独当一面干活的时候就把我调走到另外一个更加“紧急”的地方吧，那我究竟是什么东西呢？呼之来唤之去，我需要一块更加垂直的领域去深钻，我笃信这并不是一件容易的事情，就现状而言！

说到底，还是失去了陌生的感觉，身边的事物都这么熟悉如常、一成不变的，仿佛它们一出生就待在了对应的位置上，不惧怕衰老，你感觉到安全，不再提心吊胆，也就没有在进一步的必要，人是从这里开始完蛋的。

我提起过这件事情，可能只是随口提了一嘴：嘿，请你不要稀里糊涂地成为耗材！起初还洋洋得意自己能够多快好省地提升、学啊，在短短几年内过上一个“相对好”的人生。最后还是掉入了这个名为“熟悉”的漩涡，在一个地方待久了，人是会坏掉的，脑袋也会变得全部都是锈迹斑斑，我没这么乐观，时间不多了。

扯远了…

最近收到很多旅行VLOG的推送，可能平台软件也偷听到了我的小秘密吧，提醒我该去旅行放松一下了（笑。以前很流行一种说法，叫做来一场说走就走的旅行。听起来稍微老掉牙了些，很多人觉得，如果你当前的生活糟糕透了，一场意料之外的旅行，能够帮助你重启人生！

诸如这类荒唐至极的说法，在当时风靡，被许多人所信奉，我也不例外。直到那次大理的旅行，过程虽然很美好，当然你觉得自己已经好了的时候，已经变得足够勇敢、不再害怕重新面对自己的生活的时候，回到自己居住的地方，看到的依旧只是满地的鸡毛，明天，生活上的问题仍然会如期而至。生活不会因此而变得更好、或者好特别多（这只是资本和商人赚钱的手段罢了。

但我依然会对视频当中的许多地点感到神往（其实我也不再年轻，写字也不文青了，有时候文字就是有种魔力：

比起厚厚的白雪，更让我印象深刻的是圣彼得堡的灯光。

在一天只有不到6个小时白天的季节里，这里整夜整夜亮着五颜六色的霓虹灯，在为了防风而砌的厚厚的斑驳的墙上闪着光。

街上的人神情严肃的来去在像童话一样的世界里，似乎一边通过节约笑容省着电，一边又在通过这些温暖的色彩充着电，作为这个新年时节，对抗严寒的孤独的方式。

——kikkokilling

2026/3/7 - 枯木逢几春

hey! hi! how’re you doing!

太久没回来了，都快要忘记这里。

这半个月的时间，仿佛开启了飞行模式，屏蔽了许多讯息，屏蔽了许多人、事。甚至连自己原本所在的中心都快要找寻不到，回过头来看一看这段时光，我可能也会不禁感慨，这家伙真是个不折不扣的工作天才呢（笑。最近工作状态正如我先前所说，接近崩溃的边缘，其实我不知道自己能够做到多少，尽全力也想要做好…你知道的，往往事与愿违，我不太想多说什么话，很多事情也记不太清了，直到我做梦的时刻梦见工作内容，如影随形，我意识到，是时候需要给自己一些空间做调整…

俗话说得好，你想要摆脱某些东西，不是刻意不去做，而是往自己的作息里塞入更多的时间去做另外一件事情。

这两周的时间里，刚好因为项目需要，我手头正缺一款衬手的ARM脱机下载器，于是乎我好似从一片湖泊坠入另外一个深不见底的汪洋大海…（笑。

扯远了，这款嵌入式小工具就快设计好了，大概四五月份，能够和大家见面！

…

然后，我想说说新年的事情。

是的，二六年的新年，我一个人过。

找到了一片可以暂时驻足喘息的栖息地，今年一个人过年，安逸宁静。

春节在我心中渐渐要淡出去成为一个平常的日子，平常的假期就应当平常地度过。这份很临时决定的如常，仿佛让我重新找到，那颗丢失很久的心灵地图…
