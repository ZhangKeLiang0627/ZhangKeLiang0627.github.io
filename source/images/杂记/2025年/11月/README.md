我现在对于书籍的阅读量特别少，几乎不读书了，现在读的最多的文字是代码，输出的最多的也是代码，一个月下来也算有小半本合页本了吧我想（笑。似乎听起来有点落寞；所以，关于程序员到底需不需要文学修养这件事，我没什么头绪...

为什么我会突然聊到书籍？这大概是矫情的话端的开始，你本来对某样东西本来就不是很熟悉，书籍...我眼中的自由的象征，也可能是从小被教育的「心中对自由的向往」，我记得小学有堂课叫做“人类不能没有书籍”，里面提到一个假设：假如你要余生都在海上漂泊，就生活在一只小船上，你只能在上船的时候带一样东西，你会带些什么？

大部分孩子天真的回答总是天马行空的，但是最终都会在老师的引导下收敛成「带上书籍」。书籍是朋友，是人类进步的阶梯，这些名言名句如今我还是能够脱口而出，啊哈，看得出是影响很深了（笑...

想说的话在脑子里盘旋，混乱到找不到出口，乱糟糟的纠缠在一块，怎么也理不清楚。七八九月份，天气似乎着了魔，失控地反常地，一直不停的下雨，氤氲的水汽蒸发，和热浪的气旋总是会不经意间扑过来，每天起床我都在抱怨：怎么老是下雨？似乎被困在这潮湿的雨季，做什么事情都黏黏糊糊的，大大小小的事情顾不过来了会有问题出现，总算是尽力去完成，没有出什么纰漏（。

连续不断的工作，加班已经来到了第三个月，本来放假前信誓旦旦地拍着胸脯说这次国庆长假八天，我要全勤！我要三倍工资！经过前四天的加班工作，心理防线终于在第五天彻底坍塌，想着给自己放一天假吧，下午老大的信息就找过来说今天有没有坐班？这可尴尬住了，他说：你也跟着加班好几天了我都有看见，本来后面几天没你什么事就安心在家休息一下，假期是需要呼吸的，好巧不巧今天部门老总过来查班，我们组又没人（此处配一个尬笑的emoji...

假期需要呼吸。

组里最近来了不少新人，慢慢开始壮大起来，而我也慢慢变成组里资历老人了，老大说上周刚进组的新人昨天跑了，一边无奈地说着一边给我们递月饼：给，你们多吃点，这是那个走掉的人的中秋礼盒，我们组内部分了。我们其实都蛮吃惊的，急忙问原因，为什么啊？「因为他看到我们这种加班情况，害怕了，所以跑路了」...

原来，人家的前公司是小米，好不容易跳槽到这里，以为能够轻松点养老点，又被我们的加班频次吓到了，相当于从一个火坑跳到了另外一个火盆当中。

可能是从考研无日无夜的煎熬中爬出来过，我不觉得加班是一种非常辛苦的活，然后还有钱拿，那就更加无所谓，加就加吧。我们老大也会跟着加，像他这种级别的人物，入职就相当于把自己的时间都卖给了公司，既要敲代码还要向上管理、各种开会、各种群、各种事都得找他由他负责、还得应付各种饭局...似乎他就住在公司一样，他说，有时候早上六点就来到公司，晚上要开会到十点多才能下班。我问为什么早上要这么早来呢。因为这个时间点才有空写代码...

...但是人的下半辈子不能总在无休止的加班当中度过，也许我现在的工资只有五千，要是赚到七千，我就能松一口气；挣到一万的时候，生活会顺心不少，两万的时候，我就能安定下来，五万的时候，可能在想着需要赚更多的钱人生才能幸福吧。为了工作你已经牺牲了什么？睡眠、友谊、落日、亲情、健康、、、你发现了吗？

有时候，并不是挣得越多越自由，我的朋友。
