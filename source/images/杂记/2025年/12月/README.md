### 写在前面

{% note info %}
**废话不多说了，圣诞快乐🎅🎅🎅🎄！**
{% endnote %}

...

时光飞逝，真是时光飞逝，我总是喜欢在每一回杂记的开头都嘟囔上这几句。你看，十二月就在眼前，近在咫尺的气息，仿佛能够嗅到一些、新的年轻，新的思想，新的人群，似乎我已经想好来年春天是什么景色。一些风景，一些不那么重要的、不是必须的风景，成熟后，发觉路过太多的人物和事物，从前的必要变得不那么重要了...

w不知从什么地方读来一句话......（呃，老实说写到这里已经是过了两天后了，我那天晚上参加完部门聚会回来，酒后头昏的不行，本来想说写几段提神醒脑，没想到再睁眼已经天亮躺在床上了，实在是想不起来这到底是一句什么话了，哈哈哈哈哈，想起来再来说吧！

十二月，抵达，降临，到站，迈着轻盈的步伐，我对这个月份总是保持着特别的情愫，可能是过冬天了吧，可能是圣诞吧，可能是到了一年的末尾了吧，可能不用开空调省电费吧，我不能说，但我爱。
