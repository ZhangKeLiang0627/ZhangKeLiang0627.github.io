学习需要用到HomeAssistant啦，这个词其实对我来说并不陌生，但是从前都只是只听其名不见闻其声，如今终于要上手啦！拯救落灰树莓派行动！出发！

环境

  • 硬件：Raspberry Pi 4B
  • 镜像版本：2022-09-22-raspios-bullseye-arm64.img：-> 上交的镜像源
  • Python版本：3.9.2

本篇文章将简述如何迅速地在树莓派部署 HomeAssistant。
